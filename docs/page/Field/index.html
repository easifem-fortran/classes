Field

Introduction

In FEM, we use variables of different ranks, like scalar, vectors, and so on. These variables will be designated as the field. These fields can be defined at:

  • Spatial nodal points
  • Spatial-temporal points
  • Quadrature points inside the element

In addition, global matrices can also be described as the field. In this way, Fields are high level objects in finite element modeling.

Implementation of these fields depends upon the engine, like NATIVE_SERIAL, NATIVE_OMP, NATIVE_ACC, NATIVE_MPI, PETSC, LIS, among others.

Structure of fields

  • AbstractField_ defines an abstract class.
  • This abstract class will be extended to AbstractNodeField_, AbstractElementField_, AbstractMatrixField_
