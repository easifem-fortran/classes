@brief Module defines an object for representing an XML file.

This module reads an XML file and stores the element information as character information in a doubly-linked list/tree. Outside of the initialization the only other functionality is for traversing the elements. This type is basically used by the parameter list type to provide an alternative way of initializing a parameter list. ++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++!


Uses

    • Strings
    • FileType_Base
