The developement of this module is inspired from the FileType_Fortran.f90 of Futility package. The original source is located at https://github.com/CASL/Futility/blob/master/src/AbstractFile_.f90. The original code has been modified as per the code-standard of easifem library.

The Fortan file type is an extension of the abstract AbstractFile_ It provides a simplified interface to the native Fortran file capabilities and includes error checking.
