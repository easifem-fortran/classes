The developement of this module is inspired from the FileType_Base.f90 of Futility package. The original source is located at https://github.com/CASL/Futility/blob/master/src/FileType_Base.f90.

The original code has been modified as per the code-standard of easifem library.

This type is an abstract type, so it has no specific implementation. It exists only to provide a base for the extended types. It specifies the maximum lengths for the file path, name, and extension, some basic attributes of a file such as whether or not it is open and also if it is open for reading or writing are provided. Methods to interface to all attributes are also provided.

Since this is an abstract type, it has no specific implementation so see one of it’s extended types for examples on how it should be used.
