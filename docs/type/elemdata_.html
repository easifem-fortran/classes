integer(kind=I4B), private, ALLOCATABLE:: boundaryData(:)

If iel is boundary element; then Vec=BoundaryData( LBndyIndex(iel) ) contains boundary data, where vec(1) is equal to iel, and vec(2:) are ids of local facets which are boundaries of mesh
integer(kind=I4B), private :: elementType =INTERNAL_ELEMENT
integer(kind=I4B), private :: globalElemNum =0
integer(kind=I4B), private, ALLOCATABLE:: globalElements(:)

Contains the information about the element surrounding an element Lets us say that globalElem1, globalElem2, globalElem3 surrounds a local element ielem (its global element number is globalElem), then globalElements( [1,2,3] ) contains globalElem1, pFace, nFace globalElements( [4,5,6] ) contains globalElem2, pFace, nFace globalElements( [7,8,9] ) contains globalElem3, pFace, nFace Here, pFace is the local facet number of parent element globalElem (ielem) which is connected to the nFace of the neighbor element All element numbers are global element number
