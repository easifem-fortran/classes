type, public, extends(AbstractFile_) :: HDF5File_

Reading from HDF5 binary files. As implemented, there are three modes for accessing a file can be opened as: - ‘read’ - ‘write’ - ‘new’. Read mode opens an existing HDF5 file and allows the client code to interrogate that file and extract data without permissions to alter the file. Write mode opens an existing HDF5 file and allows the client code to alter its contents. New mode overwrites any file by the same name and starts from scratch.
