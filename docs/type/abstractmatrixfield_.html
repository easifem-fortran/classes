type, public, ABSTRACT, extends(AbstractField_) :: AbstractMatrixField_

Introduction

This abstract class is designed to handle the global tangent matrices defined for a computation domain. This abstract class will be extended to

  • MatrixField_
  • MPIMatrixField_
  • PetscMatrixField_
  • LISMatrixField_
  • OMPMatrixField_
  • ACCMatrixField_
