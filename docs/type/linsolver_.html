type, public, extends(AbstractLinSolver_) :: LinSolver_

Introduction

LinSolver_ data type is a container around Yusef Saad’s SparseKit lib. It is used to solve the linear system with sparse matrices

  • Reference : https://www-users.cs.umn.edu/~saad/software/SPARSKIT/
  • This class interface sparsekit and EASIFEM
